## 2. Introduction

The Courier Tracking System is a Java-based application developed to manage courier records and track parcel deliveries. It helps administrators store shipment information, update delivery status, and search parcels using tracking numbers.

---

## 3. Objectives

* Automate courier management.
* Track parcels efficiently.
* Maintain customer and shipment records.
* Reduce manual work and errors.
* Improve delivery monitoring.

---

## 4. Problem Statement

Manual management of courier records can cause delays, errors, and difficulty in tracking shipments. This system provides an automated solution for managing courier information and monitoring parcel status accurately.
